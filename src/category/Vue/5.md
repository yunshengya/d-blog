## 5.conf 配置文件修改之后，可以在 nginx 的安装目录启动 nginx.exe,也可以使用命令，start nginx 启动项目。

---

最后通过配置文件中的前端服务器地址和前端端口访问 vue 项目即可。

常用命令

```
start nginx 启动

nginx -s reload  刷新

tasklist /fi "imagename eq nginx.exe"  查看所有的nginx进程

taskkill /fi "imagename eq nginx.exe" /f  停止所有nginx进程
```
